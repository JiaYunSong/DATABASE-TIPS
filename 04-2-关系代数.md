## 集合运算符

| 运算符        | 含义     |
| ------------- | -------- |
| ∪        | 并   |
| -             | 差       |
| ∩             | 交       |
| ×             | 笛卡尔积 |



## 比较运算符

| 运算符    | 含义       |
| --------- | ---------- |
| $\theta $ | 比较运算符 |
| >         | 大于       |
| ≥         | 大于等于   |
| <         | 小于       |
| ≤         | 小于等于   |
| ＝        | 等于       |
| <>        | 不等于     |
